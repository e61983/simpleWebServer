## simpleWebServer
Just a simple web server for testing

## Auth

- Yuan

## How to build

```bash
make
```

## How to use

Default using 8080 port and using './webroot' as root.

```bash
release/simpleWebServer
```

Use your port and path.

```bash
release/simpleWebServer -p ${CUSTOM_PORT} -path ${CUSTOM_PATH}
```
